Wrapper for preparing and executing a PDOStatement.

Exceptions
Exceptionthrown if internal PDO exception is thrown
Returns
true if execution is successful (and fetchResponse is false), or an array of associative arrays of results, if fetchResponse is true.
Parameters
$sqlSQL statement to send (with question mark placeholders).
$paramsData for the placeholders. Default is an empty array.
$fetchResponseIf true (default is false), then a fetch will be done, and a response returned.
