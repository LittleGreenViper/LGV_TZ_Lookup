Detailed Description

This basic implementation of a listener simply constructs an in-memory representation of the JSON document, which is a little silly since the whole point of a streaming parser is to avoid doing just that. However, it can serve as a starting point for more complex listeners, and illustrates some useful concepts for working with a streaming-style parser.
